To run NetworkOps_Platform on your Windows computer, make sure you have:

- At least 4 GB of RAM
- 2 GHz dual-core processor or better

## 🔧 Basic Setup and Use

Once installed, here is how to begin managing your network:

1. **Open NetworkOps_Platform:**

   Find the app in your Start menu or on your desktop and open it.

2. **Connect your devices:**

   Use the menu to add network devices. You will enter details like IP address, device type, and login info.

3. **Explore tools:**

   The platform includes 178 tools for different tasks. These cover configuration management, monitoring, failure fixes, and detecting unwanted changes.

4. **Use the dashboard:**

   The web dashboard updates in real time. You can see the health and status of your network at a glance.

5. **Self-healing agents:**

   Agents run automatically to fix known problems based on set policies. You can turn these on or off in the settings.

## 🌐 Accessing the Web Dashboard

## 🛠 Troubleshooting Tips

- **Installer won’t start:**  
  Check if your antivirus or Windows Defender blocked the run. Temporarily disable it and try again.

- **Program crashes on launch:**  
  Restart your PC. Make sure all Windows updates are installed.

- **Can’t add devices:**  
  Double-check the device details (IP, type, credentials). Ensure devices are reachable on your network.

- **Dashboard won’t load:**  
  Confirm you are connected to the right network and use the correct address (`http://localhost:8080`).

- **Update problems:**  
  Try uninstalling the old version before installing the new one.

## 🔄 Common Tasks

- **Add a new device:**  
  Go to Devices > Add Device, then fill in details.

- **Run a network scan:**  
  Use Tools > Network Scan to check device status.

- **Check drift detection reports:**  
  Reports > Drift Detection displays any configuration changes.

- **Review alerts:**  
  Alerts tab shows issues requiring attention.

- **Enable self-healing:**  
  Settings > Agents, toggle self-healing mode.
